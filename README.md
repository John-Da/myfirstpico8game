# Pico-8 Demo Game

This is a small **Pico-8 test demo** where the player can move around using the arrow keys and press `X` to play a coin collection sound. It's designed as a basic experiment to test movement, input, and audio in Pico-8.

## Gameplay

- Move the character using the arrow keys.
- Press `X` to collect coins and trigger a sound effect.
- Explore the simple test environment freely.

## Getting Started

1. Install Pico-8.
2. Open the `.p8` file in the Pico-8 editor.
3. Run the game using `RUN` in the console.
